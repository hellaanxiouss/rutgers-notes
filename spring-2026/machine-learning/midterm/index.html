Formula Cheatsheet Key equations only
Perceptron Update
w ← w + yₙ·xₙ
Only when sign(wᵀxₙ) ≠ yₙ
Linear Regression
wLIN = (XᵀX)⁻¹Xᵀy = X†y
Ein = (1/N)||Xw − y||²
Closed-form, minimize squared error
Logistic / Sigmoid
θ(s) = 1 / (1 + e^(−s))
Range: (0,1) · θ(0) = 0.5
Maps any score s to a probability
Logistic Regression Error
Ein = (1/N) Σ ln(1 + exp(−yₙwᵀxₙ))
Cross-entropy loss, minimized via gradient descent
Gradient Descent Update
w(t+1) = w(t) − η·∇Ein(w(t))
η = learning rate. Too large → diverge. Too small → slow
Regularized Error (Ridge)
Eaug(w) = Ein(w) + (λ/N)||w||²
λ large → simpler model. λ=0 → no regularization
Hoeffding's Inequality
P[|Ein − Eout| > ε] ≤ 2e^(−2ε²N)
More data N → better generalization guarantee
Bayes' Theorem
P(A|B) = P(B|A)·P(A) / P(B)
Posterior = likelihood × prior / evidence
Bayes Optimal Classifier
g*(x) = argmax_y P(y|x)
Minimizes expected 0/1 loss
